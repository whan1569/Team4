파티클 필터 (Particle Filter)

## 개요

    파티클 필터는 시뮬레이션 기반의 예측 기술로, 
    연속적으로 들어오는 정보를 오차가 존재하는 관측값만을 
    가지고 예측하는 데 사용되는 알고리즘입니다. 
    이는 비선형, 비가우시안 시스템에 적합한 비모수적(non-parametric) 
    방법으로, 계속적인 몬테카를로 방법(Sequential Monte Carlo)이라고도 불립니다.

## 특징

1. 비모수적 접근: 
    파티클 필터는 사후 확률을 유한한 숫자의 파라미터들로 근사화합니다.

2. 샘플 기반: 
    사후 확률 분포를 랜덤하게 추출된 상태 샘플(파티클)의 집합으로 표현합니다.

3. 다중 모드 분포 표현: 
    복잡한 확률 분포를 표현할 수 있어 다양한 상황에 적용 가능합니다.

4. 베이즈 모델 추정: 
    주로 잠재변수가 마르코프 연쇄로 연관된 베이즈 모델을 추정하는 데 사용됩니다.

## 동작 원리

    파티클 필터의 기본 알고리즘은 다음과 같습니다:

1. 초기화: 
    초기 파티클 집합을 생성합니다.

2. 예측: 
    각 파티클을 시스템 모델에 따라 이동시킵니다.

3. 가중치 계산: 
    각 파티클의 중요도(importance factor)를 계산합니다.

4. 리샘플링: 
    가중치에 따라 파티클을 재추출합니다.

5. 추정: 
    파티클 집합을 기반으로 시스템 상태를 추정합니다.

## 응용 분야

1. 객체 추적: 
    복잡한 환경에서 움직이는 객체를 추적하는 데 효과적입니다.

2. 로보틱스: 
    SLAM(Simultaneous Localization and Mapping) 등에 활용됩니다.

3. 컴퓨터 비전: 
    실시간 감시 시스템, 영상회의 시스템 등에 사용됩니다.

4. 인간-컴퓨터 상호작용(HCI): 
    사용자의 움직임을 추적하는 데 활용됩니다.

## 장단점

장점:
    - 비선형, 비가우시안 시스템에 적용 가능
    - 복잡한 확률 분포 표현 가능
    - 확장 칼만 필터(EKF)나 무향 칼만 필터(UKF)보다 정확할 수 있음

단점:
    - 충분한 수의 샘플(파티클)이 필요함
    - 계산 복잡도가 높을 수 있음

## 결론

    파티클 필터는 복잡한 시스템의 상태 추정에 유용한 도구로, 
    특히 비선형성이 강한 문제에 효과적입니다. 
    다양한 분야에서 활용되고 있으며, 향후 더 많은 응용 분야에서 
    중요한 역할을 할 것으로 예상됩니다.
